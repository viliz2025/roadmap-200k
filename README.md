# Кибер-карта Roadmap 200k

Интерактивная дорожная карта пути от первой мини-игры до портфеля браузерных игр и цели **200 000 ₽ в месяц**.

Опубликованная версия карты:  
https://viliz2025.github.io/roadmap-200k

## Что это

Это персональная интерактивная карта прохождения проекта по созданию маленьких браузерных игр.

Карта помогает отслеживать:

- этапы движения к цели;
- статус прохождения уровней;
- ежедневные действия внутри каждого уровня;
- ссылки на результаты;
- метрики;
- отчёты;
- личные награды за прохождение уровней.

## Главная цель

Дойти от первой опубликованной мини-игры до системы, где портфель маленьких браузерных игр может приносить **200 000 ₽ в месяц**.

## Структура карты

В карте 8 уровней:

1. Одна опубликованная игра + реальные метрики  
2. Понять цифры  
3. Улучшить первую игру  
4. Сделать 3–5 игр  
5. Выявить игру-лидера  
6. Собрать портфель 10–20 игр  
7. 1–2 игры дают основной доход  
8. Система, а не случайность
